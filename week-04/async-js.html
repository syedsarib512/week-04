Asynchronous JavaScript

Asynchronous JavaScript allows you to perform tasks that take time without blocking the main thread. This is essential for creating responsive web applications. In this section, we’ll cover the basics of asynchronous programming in JavaScript.

Understanding Asynchronous Programming

Asynchronous programming allows you to perform tasks that take time, such as fetching data from an API, without freezing the user interface. JavaScript provides several ways to handle asynchronous operations, including callbacks, promises, and async/await. 

// Using a callback
function fetchData(callback) {
    setTimeout(() => {
        callback("Data received!");
    }, 2000);
}

fetchData((message) => {
    console.log(message);
});

// Using a promise
function fetchData() {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve("Data received!");
        }, 2000);
    });
}

fetchData().then((message) => {
    console.log(message);
});

// Using async/await
async function fetchData() {
    return new Promise((resolve) => {
        setTimeout(() => {
            resolve("Data received!");
        }, 2000);
    });
}

async function getData() {
    const message = await fetchData();
    console.log(message);
}

getData();

Practical Examples of Asynchronous JavaScript

Let’s see some practical examples of using asynchronous JavaScript to fetch data from an API and handle responses. 

// Fetching data from an API using fetch and promises
fetch("https://jsonplaceholder.typicode.com/todos/1")
    .then((response) => response.json())
    .then((data) => {
        console.log(data);
    })
    .catch((error) => {
        console.error("Error:", error);
    });

// Fetching data from an API using async/await
async function fetchData() {
    try {
        const response = await fetch("https://jsonplaceholder.typicode.com/todos/1");
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error("Error:", error);
    }
}

fetchData();
