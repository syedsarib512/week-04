JavaScript Fundamentals

JavaScript is a versatile programming language that allows you to create dynamic and interactive web content. In this section, we'll cover the basics of JavaScript, including its syntax, variables, data types, operators, functions, and control flow.

Basic Syntax and Structure

JavaScript code can be embedded directly into HTML pages using the <script> tag. Here's a simple example: 

<script>
    console.log("Hello, world!");
</script>

Variables, Data Types, and Operators

Variables in JavaScript can be declared using var, let, or const. JavaScript supports various data types, including strings, numbers, booleans, objects, and arrays. Operators are used to perform operations on variables and values. 

// Variable declaration
let name = "John";
const age = 30;

// Data types
let isStudent = true;
let grades = [90, 85, 88];

// Operators
let sum = 10 + 20;
let isAdult = age >= 18;

Functions and Control Flow

Functions are blocks of code designed to perform a particular task. Control flow statements, such as if-else, switch, for, and while, control the execution of the code. 

// Function declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Control flow
if (age >= 18) {
    console.log("You are an adult.");
} else {
    console.log("You are not an adult.");
}

for (let i = 0; i < grades.length; i++) {
    console.log("Grade: " + grades[i]);
}

Basic DOM Manipulation

The Document Object Model (DOM) represents the structure of an HTML document. JavaScript can interact with and manipulate the DOM to create dynamic web pages. 

// Selecting a DOM element
const title = document.querySelector("h1");

// Modifying a DOM element
title.textContent = "JavaScript Fundamentals";
title.style.color = "blue";
