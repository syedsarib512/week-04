TODO App

In this section, you will learn how to create a simple TODO application using JavaScript, HTML, and CSS. This project will help you apply the concepts learned throughout this week, such as DOM manipulation, event handling, and asynchronous programming.

HTML Structure

Start by creating the basic HTML structure for your TODO app. You will need an input field to add new tasks, a button to submit them, and a list to display the tasks. 

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>TODO App</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <header>
        <h1>TODO App</h1>
    </header>
    <main>
        <input type="text" id="new-task" placeholder="Enter a new task">
        <button id="add-task">Add Task</button>
        <ul id="task-list"></ul>
    </main>
    <script src="script.js"></script>
</body>
</html>

JavaScript Functionality

Finally, add JavaScript to handle adding, completing, and deleting tasks. Hereâ€™s a basic implementation to get you started. 

// Selecting DOM elements
const newTaskInput = document.getElementById("new-task");
const addTaskButton = document.getElementById("add-task");
const taskList = document.getElementById("task-list");

// Adding a new task
addTaskButton.addEventListener("click", () => {
    const taskText = newTaskInput.value.trim();
    if (taskText !== "") {
        const listItem = document.createElement("li");
        listItem.innerHTML = `${taskText} <button class="delete-btn">X</button>`;
        taskList.appendChild(listItem);
        newTaskInput.value = "";

        // Mark task as complete
        listItem.addEventListener("click", () => {
            listItem.classList.toggle("complete");
        });

        // Delete task
        const deleteButton = listItem.querySelector(".delete-btn");
        deleteButton.addEventListener("click", (event) => {
            event.stopPropagation();
            taskList.removeChild(listItem);
        });
    }
});
